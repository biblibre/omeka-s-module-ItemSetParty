Interface with other modules
============================

EADImport_ is a module which allows to import resources from a XML file ("ead" formatted).

When an import is performed, an item set is created with this new items.

Simply add this item set to the ItemSetParty configuration, and then check the import by browsing the tree structure.

.. _EADImport: https://github.com/biblibre/omeka-s-module-EADImport
